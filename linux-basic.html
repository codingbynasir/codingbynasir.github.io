Basic commands for linux

Open the terminal window and started to enjoy the linux

Basic commandline for linux

Command Description Example
ls It will show the directory list
mkdir It will make a directory or folder mkdir nasir It means a directory named nasir will be created
history See the recent history of terminal
sudo Means 'Super User Do' sudo add-apt-repository ppa:java8/ppa Create a repository as a super user
touch Used to make a file mkdir nasir.txt It means a file named nasir.txt will be created
chmod Means 'change file mode bits'. It will change the permission of a directory sudo chmod +r /opt/lampp read mode for lampp directory
apt stands for 'Advanced Package Tool'. Search the package and install apt-get install mplayercommand will intall the mplayer
cal This command show the calendar cal 02 1835 shows the date calendar of february 1835
cat stands for 'Concatenation' cat a.txt b.txt >> c.txtcopy the content of a.txt and b.txt. Merge them and paste into c.txt
cp stands for 'Copy' cp /home/user/Downloads abc.tar.gz /home/user/Desktop copy the abc.tar.gz from Downloads to Desktop
mv stands for 'Move' mv /home/user/Downloads abc.tar.gz /home/user/Desktop move the abc.tar.gz from Downloads to Desktop
cd stands for 'Change directory' cd /home/user/Desktop change the directory to Desktop
shutdown Switch off the system shutdown -r shutdown and restart the computer. Also use shutdown -h
